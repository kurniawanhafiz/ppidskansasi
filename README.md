# PPID SKANSASI Versi awal

Ini adalah project aplikasi **PPID (Pejabat Pengelola Informasi dan Dokumentasi)** untuk **SMK Negeri 1 Sijunjung**, dikembangkan oleh **Kurniawan Hafiz** (@justfiz) sebagai bagian dari latihan pengembangan web dinamis berbasis PHP.

## Tentang Aplikasi

Aplikasi ini berisi informasi seputar:
- **Informasi Publik Sekolah**
- **Data yang dapat diakses oleh masyarakat**
- **Halaman informasi yang mudah dikembangkan**

Aplikasi ini bertujuan untuk memberikan akses informasi publik secara sederhana, serta menjadi bagian dari portofolio pengembangan web backend.

## Teknologi yang Digunakan
- PHP
- MySQL / MariaDB
- Bootstrap (untuk layout & komponen)
- HTML & CSS
- InfinityFree (untuk hosting)

## Fitur
- Desain responsif & mobile friendly (Bootstrap)
- Tampilan informasi publik
- Struktur kode sederhana & mudah dipahami
- Konfigurasi database terpusat
